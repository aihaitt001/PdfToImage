# PDF文件转成JPG
1. 可以自己更改转成PNG
2. 可以自己调整PPI,默认为200
3. 使用Apache的PdfBox进行PDF的解析.


# A Tool For PDF To Images.
0. instal JRE.
1. make a dir "D:\pdf".
2. put your file which is PDF in it.
3. javac -jar .\PdfToImage\target\djb-1.0-SNAPSHOT.JAR
4. The images which is consolved will put in "D:\pdf";
